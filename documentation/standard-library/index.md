Swift 표준 라이브러리는 Swift 프로그램 작성을 위한 기본 기능 계층을 정의합니다.
표준 라이브러리 문서는 현재 [Apple Developer 웹사이트](https://developer.apple.com/documentation/swift/swift-standard-library)에서 호스팅되고 있습니다.

Swift 표준 라이브러리는 테스트 및 인라인 문서와 함께
[메인 Swift 저장소][swift-repo]에 포함되어 있습니다.
