## Swift Testing

Swift Testing은 표현력 있고 직관적인 API를 제공하여 Swift 코드를 쉽게 테스트할 수 있게 해주는 패키지입니다.

`#expect` 같은 매크로를 사용하여 테스트 실패 시 상세한 출력을 제공합니다. 또한 파라미터화 같은 기능으로 다양한 인자로 테스트를 쉽게 반복 실행할 수 있어, 대규모 코드베이스에서도 원활하게 확장됩니다.

XCTest로 작성된 테스트가 이미 있다면, Swift Testing으로 작성한 새 테스트와 나란히 실행할 수 있습니다. 이를 통해 자신의 속도에 맞춰 점진적으로 테스트를 마이그레이션할 수 있습니다.

Swift Testing에 대한 자세한 정보는 [GitHub 프로젝트 페이지](https://github.com/swiftlang/swift-testing)에서 확인할 수 있습니다.
