Windows 워크그룹은 Swift를 사용한 Windows 애플리케이션 개발을 촉진하는 팀입니다.

## 헌장

Windows 워크그룹의 목표는 사용자가 Swift 언어와 관련 도구를 사용하여 Windows 애플리케이션을 개발할 수 있도록 Swift의 Windows 지원을 유지 관리하는 것입니다.

Windows 워크그룹은 다음 활동을 수행합니다:

- 공식 Swift 배포판의 Windows 지원을 개선하고 유지 관리합니다.
- Foundation, Dispatch 등 핵심 Swift 패키지가 Windows 관용구에 맞게 더 잘 동작하도록 개선 사항을 제안합니다.
- Swift 프로젝트에 향후 Windows 지원 방향에 대한 제안을 합니다.
- Swift와 Windows API 간 브리징 및 Windows 애플리케이션에 Swift 라이브러리를 배포하는 모범 사례를 파악하고 제안합니다.
- Windows에서 Swift 애플리케이션의 디버깅 및 프로파일링 지원을 개선하고 유지 관리합니다.
- Windows 전용 도구 지원을 개선하고 유지 관리합니다.
- 다양한 커뮤니티 Swift 패키지에 Windows 지원을 추가하도록 조언하고 지원합니다.

## 소통

Swift Windows 워크그룹은 일반적인 논의에 [Swift Windows 포럼](https://forums.swift.org/c/platform/windows/67)을 사용합니다. Swift 포럼에서 [@windows-workgroup](https://forums.swift.org/g/windows-workgroup)으로 비공개 메시지를 보낼 수도 있습니다.

## 멤버십

Windows 워크그룹 멤버십은 기여하고자 하는 누구에게나 열려 있습니다. 멤버들은 정기 화상 회의와 Swift 포럼을 통해 소통합니다.

Windows 워크그룹은 Swift 행동 강령을 준수합니다. 커뮤니티 멤버가 워크그룹이나 그 구성원의 행동 강령 준수에 대해 우려 사항이 있으면 Swift Core Team 구성원에게 연락해야 합니다.

Platform Steering Group이 Windows 워크그룹의 의장을 선출합니다. 의장은 워크그룹에 대한 특별한 권한은 없지만, 다음을 포함하여 원활한 운영을 책임집니다:

- 정기 회의를 조직하고 주도합니다.
- 워크그룹이 커뮤니티와 효과적으로 소통하도록 합니다.
- 필요한 경우 워크그룹 대표와 다른 Swift 워크그룹 또는 팀 간 회의를 조율합니다.

워크그룹이 방향을 확신하지 못하거나 합의에 이르지 못하는 경우, 멤버는 관련 Steering Group에 안건을 상정하여 검토를 요청할 수 있습니다. 중요한 결정은 커뮤니티 참여와 Steering Group 감독을 위해 일반적인 Swift Evolution 프로세스를 따라야 합니다.

## 회의

Windows 워크그룹은 격주 수요일 오전 9시(미국 태평양 표준시)에 회의를 진행합니다.

워크그룹 회의는 공개 토론을 위한 것으로, Swift 커뮤니티 멤버라면 누구나 [@windows-workgroup](https://forums.swift.org/new-message?groupname=windows-workgroup)으로 사전에 메시지를 보내 초대를 요청할 수 있습니다.

## 커뮤니티 참여

누구나 다음과 같은 방법으로 기여할 수 있습니다:

- 설계 논의에 참여합니다.
- 포럼에서 질문하거나 답변합니다.
- 버그를 신고하거나 분류합니다.
- Windows 지원 라이브러리 프로젝트에 풀 리퀘스트를 제출합니다.
- Swift 포럼에서 아이디어를 논의합니다. [Windows](https://forums.swift.org/c/platform/windows/67) 카테고리에서 새 주제를 만들거나, 기존 주제에 windows 태그를 추가할 수 있습니다.
- Windows 경험을 개선하는 새로운 도구를 개발하거나 기존 도구를 개선합니다.
- Windows 워크그룹 멤버에게 포럼의 [@windows-workgroup](https://forums.swift.org/new-message?groupname=windows-workgroup)으로 직접 피드백을 제공합니다. 워크그룹 의장은 주요 안건과 주제를 정기 회의에서 논의하도록 가져옵니다. 워크그룹이 해당 안건에 대한 조치를 결정합니다.
- Windows 워크그룹의 정기 화상 회의에 참여합니다. [@windows-workgroup](https://forums.swift.org/new-message?groupname=windows-workgroup)으로 메시지를 보내 접근 권한을 요청하세요.
