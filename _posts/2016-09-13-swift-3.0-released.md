오픈 소스 공개 이후 첫 번째 메이저 릴리스인 Swift 3.0이 공식 출시되었습니다! Swift 3은 핵심 언어와 표준 라이브러리의 대규모 개선 및 정제, Linux 포팅의 주요 기능 추가, [Swift Package Manager]의 첫 공식 릴리스를 포함하는 대규모 릴리스입니다.

### Swift 3으로 마이그레이션

Swift 3은 소스 호환성을 깨는 릴리스이며, 이는 주로 [SE-0005](https://github.com/swiftlang/swift-evolution/blob/master/proposals/0005-objective-c-name-translation.md)와 [SE-0006](https://github.com/swiftlang/swift-evolution/blob/master/proposals/0006-apply-api-guidelines-to-the-standard-library.md)의 변경 때문입니다. 이 변경은 표준 라이브러리 API 이름뿐만 아니라 Objective-C API(특히 Cocoa)가 Swift로 임포트되는 방식도 완전히 바꿉니다. 많은 변경 사항이 기계적이지만, 일반적인 Swift 프로젝트에서는 그 수가 상당합니다.

Swift 3으로의 전환을 돕기 위해 [Xcode 8.0]에는 필요한 소스 변경 사항의 많은 부분을 자동으로 처리하는 코드 마이그레이터가 포함되어 있습니다. 또한 [마이그레이션 가이드](/migration-guide/)에서 많은 변경 사항을 안내합니다 — 특히 기계적이지 않고 더 직접적인 검토가 필요한 부분에 대해서 말입니다.

### 문서

Swift 3.0용 [The Swift Programming Language](/documentation/tspl) 업데이트 버전을 Swift.org에서 확인할 수 있습니다. Apple iBooks Store에서도 무료로 제공됩니다.

### 플랫폼

#### Linux (Ubuntu 14.04 및 Ubuntu 15.10)

Linux 포팅에 [Swift Core Libraries](/documentation/core-libraries/)와 [Swift Package Manager]가 포함됩니다.

Ubuntu 14.04와 Ubuntu 15.10용 공식 바이너리는 [다운로드](/download/)할 수 있습니다.

#### Apple (Xcode)

Apple 플랫폼 개발을 위해 Swift 3.0은 [Xcode 8.0]에 포함되어 제공됩니다.
