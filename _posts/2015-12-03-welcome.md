Swift.org 블로그에 오신 것을 환영합니다! 오늘 오픈 소스 Swift 프로젝트와 함께 Swift.org 웹사이트를 공개했습니다. 오픈 커뮤니티에서 함께 이슈를 찾고 수정하며, 개선 사항을 추가하고, Swift를 새로운 플랫폼으로 확장해 나갈 수 있게 되어 정말 기쁩니다.

이 블로그는 Swift를 개발하는 엔지니어들이 공지 사항을 전달하고 중요한 커뮤니티 주제를 다루는 공간입니다.


## 프로젝트

Swift는 훌륭한 소프트웨어를 만들기 위한 완전한 생태계를 제공하는 여러 프로젝트로 구성됩니다. Swift 컴파일러 프로젝트는 Swift 문법을 해석하고, 올바른 코드 작성을 돕는 진단 메시지를 생성하며, LLVM을 활용해 머신 코드를 생성합니다. LLDB 프로젝트는 대화형 프로그래밍을 위한 REPL을 포함하는 최고 수준의 디버거입니다. Swift 표준 라이브러리 프로젝트에는 Swift로 소프트웨어를 작성하는 데 필요한 모든 핵심 타입과 기본 기능이 포함되어 있습니다.

오늘 Swift의 오픈 소스 프로젝트 두 가지를 추가로 공개합니다: Core Libraries 프로젝트와 새로운 Swift Package Manager 프로젝트입니다.


### Swift Package Manager

[Swift Package Manager](/documentation/package-manager/)는 Swift 코드를 빌드하고 공유하기 위한 강력하고 사용자 친화적인 도구를 목표로 하는 새로운 프로젝트입니다. 컴파일된 바이너리 라이브러리보다는 소스 코드 공유에 최적화된 패키지 매니저를 만드는 데 집중하고 있습니다. 이 프로젝트는 아직 초기 개발 단계이며, Swift의 개방적이고 협력적인 프로세스를 통해 설계·개발될 예정입니다.

[GitHub의 Apple 홈](http://github.com/apple)에서 예제 패키지 저장소와 패키지 매니저의 소스 코드 및 추가 정보를 확인할 수 있습니다.


### Core Libraries

[Swift Core Libraries](/documentation/core-libraries/) 프로젝트는 Swift 표준 라이브러리 위에 구축된 상위 수준의 API 세트입니다. 이 라이브러리는 지역화, 네트워킹 기본 요소, 단위 테스트, 사용자 환경설정 등의 기능을 제공합니다. 또한 더 많은 Swift 코드를 작성하고 새로운 패키지를 만들 때 유용한 코딩 규칙도 소개합니다.

Core Libraries는 Apple 플랫폼에 포함된 프레임워크, 즉 Foundation, libdispatch, XCTest를 기반으로 합니다. 이 프레임워크의 Swift 오픈 소스 버전은 여러 플랫폼에서 동일한 Swift 코드를 일관된 기능으로 쉽게 사용할 수 있도록 하기 위해 만들어졌습니다.


## Swift.org 웹사이트

이 웹사이트는 Swift 프로젝트의 홈으로, 커뮤니티에 참여하는 데 필요한 리소스 링크를 제공합니다. 사이트를 둘러보시길 권하지만, 첫 번째 게시물에서 몇 가지 핵심 링크를 소개합니다:


* [GitHub의 Apple 홈](http://github.com/apple)에 모든 Swift 소스 코드가 있습니다
* [Swift 메일링 리스트](/community/#mailing-lists)를 통해 소통합니다
* [시작하기](/getting-started/) 페이지에서 Swift 개발 환경 설정을 안내합니다
* [다운로드](/download/) 페이지에서 지원 플랫폼용 사전 빌드 바이너리를 받을 수 있습니다
* [Swift Evolution 프로세스](/contributing/#evolution-process)에서 새로운 기능이 어떻게 제안되는지 확인할 수 있습니다


오픈 소스 Swift 커뮤니티에 오신 것을 환영합니다.
