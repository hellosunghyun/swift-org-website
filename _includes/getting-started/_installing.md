## Swift 설치

Swift 여정을 시작하려면, [Swift를 설치](/install)하여 **macOS**, **Linux**, 또는 **Windows**에서 사용해 보세요.

> 팁: Swift가 설치되었는지 확인하려면 셸 또는 터미널 앱에서 `swift --version`을 실행하세요.

Swift에는 Swift 코드 배포를 관리하는 [Swift Package Manager(SwiftPM)]({% link documentation/package-manager/index.md %})가 함께 제공됩니다. 다른 Swift 패키지를 애플리케이션과 라이브러리에 쉽게 가져올 수 있어, Swift 개발자에게 유용한 도구입니다.

Swift는 [Apache License, Version 2.0](/LICENSE.txt)에 따라 배포됩니다.
